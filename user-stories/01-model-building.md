# 算例构建与配置场景

## US-001: 从模板创建新算例 🟢

### 用户角色
电力系统研究员，需要快速搭建一个标准测试系统

### 目标
基于CloudPSS提供的IEEE标准模板，创建一个新的算例用于研究

### 场景描述
用户正在进行新型电力系统稳定性研究，需要一个IEEE 39节点系统作为基础。
1. 浏览CloudPSS公开的IEEE标准算例库
2. 找到IEEE 39节点（10机New England）系统
3. 复制该算例到自己的工作空间，命名为"2024稳定研究_基础算例"
4. 确认算例复制成功，可以正常打开

### 技能要求
- `modelManagement.listModels()` - 搜索公开算例
- `modelManagement.searchModels()` - 按关键词过滤
- `manage.getModel()` - 获取算例详情
- `manage.copyModel()` - 复制算例（需新增）

### 验收标准
- [ ] 能够搜索并找到IEEE标准算例
- [ ] 能够复制算例到自己的空间
- [ ] 新算例名称正确，内容完整

### 优先级
P0

## US-002: 修改发电机参数 🟢

### 用户角色
研究人员，需要调整发电机参数进行敏感性分析

### 目标
在复制的算例基础上，修改特定发电机的出力参数

### 场景描述
用户已经在"2024稳定研究_基础算例"中，需要研究不同发电出力对系统稳定性的影响。
1. 查看算例中所有发电机元件列表
2. 定位到"Generator-5"（第5台发电机）
3. 修改有功出力从500MW调整为600MW
4. 修改无功出力从100MVar调整为150MVar
5. 保存修改后的参数方案，命名为"Gen5增出力方案"

### 技能要求
- `manage.getModel()` - 获取算例结构
- `configure.configureGenerator()` - 修改发电机参数
- `manage.createConfig()` - 创建参数方案
- `manage.saveModel()` - 保存算例

### 验收标准
- [ ] 能够列出所有发电机元件
- [ ] 能够准确定位目标发电机
- [ ] 参数修改成功并保存

### 优先级
P0

## US-003: 调整负荷水平 🟢

### 用户角色
规划工程师，需要分析不同负荷水平下的系统运行状态

### 目标
批量调整系统负荷，创建不同负荷水平的参数方案

### 场景描述
用户需要分析系统在80%、100%、120%负荷水平下的运行特性。
1. 获取算例中所有负荷元件
2. 创建三个参数方案：
   - "低负荷方案"：所有负荷乘以0.8
   - "基础方案"：保持原始负荷
   - "高负荷方案"：所有负荷乘以1.2
3. 验证参数方案创建成功

### 技能要求
- `extract.extractComponents()` - 提取负荷元件
- `configure.configureLoad()` - 配置负荷参数
- `manage.createConfig()` - 创建参数方案

### 验收标准
- [ ] 能够识别所有负荷元件
- [ ] 能够批量修改负荷参数
- [ ] 三个参数方案创建成功

### 优先级
P1

## US-004: 添加新线路 🟡

### 用户角色
电网规划人员，需要在算例中新增一条输电线路

### 目标
在现有算例拓扑中添加一条新的输电线路

### 场景描述
用户研究新建线路对系统潮流的影响，需要在Bus-10和Bus-15之间添加一条新线路。
1. 获取系统拓扑信息，确认Bus-10和Bus-15存在
2. 添加一条新的输电线路元件，参数如下：
   - 名称：NewLine-10-15
   - 长度：50km
   - 电压等级：220kV
   - 电阻：0.03Ω/km
   - 电抗：0.35Ω/km
3. 连接线路到两个节点
4. 验证新线路添加成功，拓扑连接正确

### 技能要求
- `topologyAnalysis.getTopology()` - 获取拓扑
- `create.addComponent()` - 添加元件
- `topologyAnalysis.connectComponent()` - 连接元件（需增强）

### 验收标准
- [ ] 能够识别目标节点
- [ ] 新线路元件添加成功
- [ ] 拓扑连接正确

### 优先级
P1

## US-007: 从零构建简单算例 🔴

1. 创建空白算例，命名为"3节点测试系统"

3. 添加1台发电机连接到Bus1
4. 添加2个负荷分别连接到Bus2和Bus3
5. 添加2条输电线路连接三个节点
6. 设置基准容量和基准电压

### 技能要求
- `create.createBlankModel()` - 创建空白算例（需新增）
- `create.addComponent()` - 添加各类元件
- `create.setBaseValues()` - 设置基准值（需新增）
- `topologyAnalysis.validateModel()` - 验证算例

- [ ] 空白算例创建成功
