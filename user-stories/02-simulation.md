# 仿真运行与分析场景

## US-011: 快速潮流计算 🟢

### 用户角色
系统分析工程师，需要进行日常潮流计算

### 目标
对算例执行潮流计算并快速获取关键结果

### 场景描述
用户需要对"IEEE39基础算例"进行潮流计算。
1. 选择算例和计算方案
2. 执行潮流计算
3. 等待计算完成（显示进度）
4. 快速查看关键结果：
   - 最高/最低电压节点
   - 最大负载率线路
   - 总网损
   - 是否存在越限

### 技能要求
- `powerFlow.runPowerFlow()` - 运行潮流
- `powerFlow.getBusVoltages()` - 获取节点电压
- `powerFlow.getBranchFlows()` - 获取支路功率
- `powerFlow.checkViolations()` - 越限检查

### 验收标准
- [ ] 仿真启动成功
- [ ] 结果获取完整
- [ ] 越限状态正确识别

### 优先级
P0

## US-013: 参数敏感性分析 🟢

### 用户角色
研究人员，需要分析系统参数对潮流的影响

### 目标
扫描某参数的变化范围，分析对系统的影响

### 场景描述
用户想研究负荷增长对系统电压稳定性的影响。
1. 选择负荷增长系数作为扫描参数
2. 设置扫描范围：0.8 ~ 1.3，步长0.05
3. 执行参数扫描
4. 分析结果：
   - 绘制电压-负荷曲线
   - 识别临界负荷点
   - 计算敏感性系数

### 技能要求
- `batchEnhanced.parameterSweep()` - 参数扫描
- `batchEnhanced._analyzeSensitivity()` - 敏感性分析

### 验收标准
- [ ] 参数扫描完成
- [ ] 敏感性系数计算正确
